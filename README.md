# Snake
A classic Snake game created for Harvard's CS50 Introduction to Computer Science.  
This project was built in Scratch and demonstrates core programming concepts such as loops, collision detection, conditionals, events, and score tracking.

The objective is to survive as long as possible and achieve the highest score.
